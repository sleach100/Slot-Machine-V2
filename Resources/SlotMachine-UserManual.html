Overview

From simple grooves to mind-bending polyrhythms — S.L.O.T Machine lets you build beats in seconds, not sessions. Each slot runs on its own rhythmic pulse, letting you layer independent time signatures effortlessly. Once your groove is locked in, export straight to your DAW and drop it right into your workflow. S.L.O.T Machine isn’t just a drum toy - it’s a rhythm engine.

S.L.O.T Machine has two timing modes. Count mode and Rate mode. Most musicians will want to use Count mode, as it is more intuitive, works the way musicians think, and has a graphical visualizer, but Rate mode gives you more fine control in some situations.

Count Mode

Count Mode is designed for precise, beat-based control of rhythmic patterns.Each slot in the sequencer is divided into a specific number of beats — for example, 4, 7, or 13 — and the sample in that slot will trigger once per beat, cycling through its full loop before starting again.

How It Works

  • Each slot’s Count value determines how many beats occur in one loop cycle. A count of 4 will create a simple 4-beat rhythm; 7 will form a repeating 7-beat polyrhythm.
  • The sequencer synchronizes all slot cycles to the Master BPM, aligning them so the full polyrhythmic pattern resolves at the least common multiple of all slot counts.
  • You can visualize this in the Visualizer — each slot becomes a polygon with a number of sides equal to its Count. The moving bead represents that slot’s playback phase.
  • The Count Beat Mask Grid lets you toggle individual beats on or off within the cycle. For instance, in an 8-beat slot, you can mute beats 3 and 7 to create syncopation. This powerful feature should not be overlooked.
  • All beat masking is represented internally by a 64-bit pattern mask, allowing complex rhythmic gating.

When To Use

Use Count Mode when you want to:

  • Lock samples to fixed subdivisions of the main tempo.
  • Create clean, repeating polyrhythms (e.g. 3 vs 4 vs 5).
  • Manually shape rhythmic phrasing using per-beat masking.

Rate Mode

Rate Mode offers ratio-based rhythmic control — perfect for free-form or evolving polyrhythms.Instead of counting beats, each slot plays according to a Rate multiplier relative to the master tempo.

How It Works

  • The Rate parameter defines how quickly a slot cycles compared to one full master beat. A rate of 1.0000 plays once per beat, 2.0000 plays twice per beat, and 0.5000 plays once every two beats.
  • Unlike Count Mode (which divides beats), Rate Mode multiplies or divides the timing continuously, allowing more fluid tempo relationships — even fractional values like 1.3333 or 2.7500.
  • Slots still follow mute, solo, volume, and decay controls — but in Rate Mode, the rhythmic overlap between slots can become more complex and evolving.
  • Each rate interacts dynamically with the Master BPM, so changing BPM will proportionally stretch or compress all slot timings.

When To Use

Use Rate Mode when you want to:

  • Explore non-integer polyrhythms and evolving textures.
  • Create ratios like 3:2, 7:5, or golden-ratio-style rhythmic loops.
  • Design more fluid or “off-grid” grooves that still sync to the master tempo.

Starting the App

  • Launch the executable directly.
  • On startup, S.L.O.T Machine initializes with a blank pattern and 120 BPM setting.

Pattern Management

  • Multiple pattern tabs let you create and organize different grooves.
  • Right-click the tab bar to rename, duplicate, create, or import new patterns.
  • You can save the current pattern’s state and recall it instantly. SAVE will save all tabs. LOAD will restore all tabs in the saved file.
  • IMPORT lets you import a saved pattern from a file. If there’s more than one tab saved, you can choose which tab to import.
  • Switching patterns while playing will finish the current pattern, then begin playing the newly selected tab.

Exporting

MIDI Export

  • Generates a Type 1 MIDI file with up to 16 channels — one per slot.
  • Each slot outputs a single note (C5, MIDI note 60) per active beat.
  • Great for transferring grooves to DAWs like FL Studio, Ableton, or Logic.

Audio Export

  • Renders your current groove to a stereo audio file.
  • Exports one or more full cycles including per-slot rate, decay, and gain settings.

Visual Options

  • Customize glow and pulse colors for the interface.
  • Adjust vertical slot row scaling (75–100%). Use if the slots won't all fit on screen.
  • Toggle display of master and slot progress bars.

Tips

  • You can manually trigger samples by clicking in the slot even when playback is stopped — handy for sound-checking.
  • Mono samples are automatically duplicated to stereo.

Technical Summary

  • Audio: Stereo output only, no input.
  • Slots: 16 independent sample players.
  • Sample Length: ~8 minutes per file.
  • Envelope: Exponential decay per slot.
  • Export: Audio (WAV) & MIDI (Type 1).
  • Framework: JUCE 7; standalone & plugin (VST3/AU).
